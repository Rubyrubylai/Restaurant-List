# 我的餐廳清單
此專案提供使用者查看及搜尋餐廳資訊，每天不用煩惱今天要吃甚麼!

# 功能列表
+ 首頁查看所有餐廳的資料(名稱、分類、評分）
+ 從首頁上的搜尋方框搜尋餐廳
+ 點選首頁的餐廳圖片，查看其詳細資訊（類別、地址、電話、評分）

# 安裝
1. 開啟終端機(Terminal)，cd到存放專案本機位置並執行:
```
git clone https://github.com/Rubyrubylai/Restaurant-List
```
2. 安裝套件
```
npm install
```

3. 執行專案
```
npm run dev
```

4. 開啟網址在本機端 http://localhost:3000
